Sawfish App Store

A fast, school-safe way to access games and student tools.

Why This Exists

Many schools rely on managed app tools that can feel slow, limited, or missing the apps students actually use. The Sawfish App Store provides a clean, modern alternative that feels like a native app store while remaining accessible through any modern web browser without requiring special permissions or software installation beyond the standard PWA installation process.

The Sawfish App Store is a progressive web application designed specifically for school environments. It has been carefully curated to include only school-appropriate content while still providing engaging entertainment and useful productivity tools. All apps in the store have been reviewed to ensure they meet school safety standards and do not contain inappropriate content, excessive advertising, or data collection practices that would raise privacy concerns.

How to Install

This works best when added to your device's Home Screen for a full app experience:

1
Tap the Share Button

In Safari or your browser, look for the share icon typically located in the toolbar. This button usually looks like a square with an arrow pointing upward on iOS devices or three dots connected by lines on Android devices.

2
Select "Add to Home Screen"

Scroll down through the share sheet options until you find "Add to Home Screen" or "Install App". Tap this option to proceed with the installation process.

3
Confirm the Installation

Review the app name and icon that will appear on your home screen. You can customize these if desired, then tap "Add" or "Install" to confirm and begin the installation process.

4
Open from Home Screen

Once installed, tap the Sawfish App Store icon from your device's home screen to open the app. The installation screen will disappear and the full application will be available.

What Installation Unlocks

When installed as a Progressive Web App, you get access to the complete feature set:

Game Hub

Access a curated collection of school-friendly games including puzzles, platformers, simulations, and creative building experiences.

Chat and call your classmates through integrated communication tools designed for quick and easy collaboration.

Enjoy a faster, more responsive experience with native-like performance, smoother navigation, and distraction-free interface.

Hack Stuff

By Sawfish Developer Group

Category: Utilities / Experimental Tools

Overview

Hack Stuff represents a comprehensive collection of advanced utilities and experimental development tools designed specifically for students, programmers, and technology enthusiasts who need access to sophisticated browser-based functionality within educational environments. This powerful toolkit provides sandboxed access to various development and debugging utilities that would typically require elevated system permissions or specialized software installations. The platform has been carefully engineered to operate within strict security boundaries while still delivering genuine utility for learning, experimentation, and problem-solving purposes.

The foundation of Hack Stuff is built upon the principle that students learn best when they have access to real tools that professionals use in the industry. Rather than providing simplified educational approximations, this collection offers genuine development utilities adapted for browser environments. Each tool within the suite has been specifically modified to work within browser security constraints while maintaining the core functionality that makes these tools valuable for learning and development work.

The user interface mimics professional integrated development environments, giving students experience with tools they will encounter in real-world development careers. The layout is intuitive yet comprehensive, presenting multiple panels and windows that can be rearranged and customized to suit individual preferences. Keyboard shortcuts and command palettes provide efficient access to frequently used functions, helping users develop workflows that transfer to professional development environments.

Key Features and Capabilities

The Hack Stuff suite encompasses an impressive array of tools serving different development and debugging needs. The HTML inspector provides a comprehensive view of any webpage's document object model, allowing users to explore element hierarchies, view computed styles, and understand how different parts of a webpage relate to each other. This tool is invaluable for students learning web development who need to see how real websites are constructed and styled.

The CSS analyzer breaks down styling rules with exceptional granularity, showing cascade interactions, specificity calculations, and inheritance patterns. Users can see exactly how browser engines resolve style conflicts, which is essential knowledge for anyone serious about mastering CSS. The tool includes visual highlighting of affected elements and real-time preview of style changes, making it an interactive learning experience rather than a passive inspection tool.

JavaScript debugging capabilities include a fully functional console with command history, variable inspection, and breakpoint management. The console supports modern JavaScript features and provides intelligent autocomplete suggestions based on the execution context. Breakpoints can be set on lines, conditions, or DOM events, with detailed stack traces and scope inspection when execution pauses. This professional-grade debugging environment helps students understand program flow and develop debugging skills applicable to any programming language.

Network request monitoring captures all HTTP traffic between the browser and servers, displaying headers, payloads, timing information, and response data. This tool is essential for understanding how web applications communicate, how APIs work, and how to diagnose connectivity issues. The interface organizes requests logically and provides powerful filtering and search capabilities for navigating large request logs.

Technical Implementation

Hack Stuff is built using modern web technologies including Service Workers for offline functionality, Web Components for modular architecture, and CSS Custom Properties for theming. The application employs progressive enhancement strategies, ensuring that core functionality works across all modern browsers while providing enhanced features where supported. Performance optimization is a priority, with careful attention to memory management and efficient rendering to ensure the tools remain responsive even when processing large amounts of data.

The sandbox architecture ensures that all tools operate safely within browser security boundaries. While users can inspect and analyze any webpage, they cannot modify the underlying pages or access超出浏览器安全模型范围的功能. This design allows for powerful analysis capabilities while maintaining the security guarantees that schools require. The sandbox also prevents tools from interfering with each other, ensuring stable operation even during complex debugging sessions.

Educational Applications

Hack Stuff serves as an exceptional educational platform for computer science curricula. Students can use the HTML inspector to understand webpage structure, learning how real websites are built from the ground up. The CSS analyzer provides hands-on experience with styling concepts that might otherwise remain abstract. By seeing how browser engines actually resolve style conflicts, students develop intuitive understanding of cascade mechanics.

Programming courses benefit from the JavaScript debugging capabilities, allowing students to step through code execution and understand program flow. The console provides an interactive environment for experimentation and learning, supporting the exploratory learning approach that research shows is most effective for programming education. Teachers can demonstrate concepts in real-time and students can verify their understanding through hands-on experimentation.

Network monitoring tools help students understand how web applications communicate, providing concrete visualization of HTTP requests and responses. This knowledge is essential for understanding modern web architecture and API design. Students can watch how web applications load resources, communicate with backends, and handle user interactions, building mental models of web system architecture.

Limitations and Considerations

While Hack Stuff provides genuine development utilities, certain limitations apply due to browser security boundaries. Users cannot modify external websites, access restricted system resources, or perform actions that would violate browser security policies. These limitations are intentional and necessary for safe operation in school environments.

Some advanced debugging features available in native development environments are not available in browser-based tools. Memory profiling is limited, and certain low-level system interactions are not possible. However, for educational purposes and front-end development, the available tools provide excellent coverage of essential functionality.

Draw a Circle

By Sawfish Developer Group

Category: Games / Skill

Overview

Draw a Circle is a deceptively simple yet endlessly engaging reflex and precision challenge that tests your ability to create the most perfect circle possible using only your finger or mouse. The game tracks the geometry of your drawing in real-time, analyzing every curve and angle to determine how close you came to mathematical perfection. Higher scores require smoother, more consistent movements without hesitation or jagged corrections.

The appeal of Draw a Circle lies in its immediate feedback and universal accessibility. Anyone can understand the goal instantly, and the challenge scales naturally with skill. Beginners can achieve decent scores while learning to control their drawing implement, while experts can pursue increasingly precise perfection. The competitive element comes from personal improvement and community leaderboards showing the highest precision scores.

Features and Gameplay

The scoring algorithm analyzes multiple aspects of your drawing including circularity, consistency of radius, smoothness of curve, and completion time. These factors combine to produce a score out of 100, with perfect circles receiving scores approaching 100 and highly irregular drawings scoring accordingly lower. The algorithm is calibrated to reward genuine circularity while penalizing artificially achieved perfect shapes.

Multiple game modes provide varied experiences. Practice mode allows unlimited attempts without score pressure, perfect for learning and improvement. Time attack mode challenges players to achieve the highest score within a time limit. Challenge mode presents specific targets that must be achieved to progress. Daily challenges provide community-wide targets for shared competition.

Sandboxels

By R74n

Category: Games / Simulation / Physics

Overview

Sandboxels represents a remarkable achievement in browser-based simulation, offering an almost infinitely expandable physics sandbox with over 500 unique elements that interact according to realistic physical rules. Falling sand simulations have long fascinated programmers and players alike, but Sandboxels elevates the concept to unprecedented heights with its comprehensive element library, sophisticated interaction rules, and smooth performance in browser environments. The simulation provides hours of creative play while simultaneously teaching scientific concepts in an engaging, hands-on manner.

Element Categories

Basic materials form the foundation of Sandboxels exploration. Sand, dust, and powder follow traditional falling sand physics, pooling and flowing based on particle interactions. Liquids range from water with its characteristic flowing and pooling behavior to viscous substances like honey and oil with different movement characteristics.

Temperature-based elements introduce dynamic change. Fire burns organic materials, spreads to adjacent burnable elements, and produces smoke. Ice forms when water reaches freezing temperatures and can melt when heated. Steam rises when water boils, creating atmospheric effects. These temperature mechanics create realistic phase transitions.

Overview

Paper Io 2 delivers addictive territory conquest gameplay where players control characters that leave trails as they move, competing to claim territory by enclosing areas and returning to their base to fill them in. The game combines quick reflexes with strategic territorial thinking, creating competitive matches that are easy to learn but challenging to master.
