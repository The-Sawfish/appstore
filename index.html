Sawfish App Store

A fast, school-safe way to access games and student tools.

Why This Exists

Many schools rely on managed app tools that can feel slow, limited, or missing the apps students actually use. The Sawfish App Store provides a clean, modern alternative that feels like a native app store while remaining accessible through any modern web browser without requiring special permissions or software installation beyond the standard PWA installation process.

The Sawfish App Store is a progressive web application designed specifically for school environments. It has been carefully curated to include only school-appropriate content while still providing engaging entertainment and useful productivity tools. All apps in the store have been reviewed to ensure they meet school safety standards and do not contain inappropriate content, excessive advertising, or data collection practices that would raise privacy concerns.

How to Install

This works best when added to your device's Home Screen for a full app experience:

1
Tap the Share Button

In Safari or your browser, look for the share icon typically located in the toolbar. This button usually looks like a square with an arrow pointing upward on iOS devices or three dots connected by lines on Android devices.

2
Select "Add to Home Screen"

Scroll down through the share sheet options until you find "Add to Home Screen" or "Install App". Tap this option to proceed with the installation process.

3
Confirm the Installation

Review the app name and icon that will appear on your home screen. You can customize these if desired, then tap "Add" or "Install" to confirm and begin the installation process.

4
Open from Home Screen

Once installed, tap the Sawfish App Store icon from your device's home screen to open the app. The installation screen will disappear and the full application will be available.

What Installation Unlocks

When installed as a Progressive Web App, you get access to the complete feature set:

Game Hub

Hack Stuff

By Sawfish Developer Group

Category: Utilities / Experimental Tools

Overview

Hack Stuff represents a comprehensive collection of advanced utilities and experimental development tools designed specifically for students, programmers, and technology enthusiasts who need access to sophisticated browser-based functionality within educational environments. This powerful toolkit provides sandboxed access to various development and debugging utilities that would typically require elevated system permissions or specialized software installations. The platform has been carefully engineered to operate within strict security boundaries while still delivering genuine utility for learning, experimentation, and problem-solving purposes.

The foundation of Hack Stuff is built upon the principle that students learn best when they have access to real tools that professionals use in the industry. Rather than providing simplified educational approximations, this collection offers genuine development utilities adapted for browser environments. Each tool within the suite has been specifically modified to work within browser security constraints while maintaining the core functionality that makes these tools valuable for learning and development work.

The user interface mimics professional integrated development environments, giving students experience with tools they will encounter in real-world development careers. The layout is intuitive yet comprehensive, presenting multiple panels and windows that can be rearranged and customized to suit individual preferences. Keyboard shortcuts and command palettes provide efficient access to frequently used functions, helping users develop workflows that transfer to professional development environments.

Key Features and Capabilities

The Hack Stuff suite encompasses an impressive array of tools serving different development and debugging needs. The HTML inspector provides a comprehensive view of any webpage's document object model, allowing users to explore element hierarchies, view computed styles, and understand how different parts of a webpage relate to each other. This tool is invaluable for students learning web development who need to see how real websites are constructed and styled.

The CSS analyzer breaks down styling rules with exceptional granularity, showing cascade interactions, specificity calculations, and inheritance patterns. Users can see exactly how browser engines resolve style conflicts, which is essential knowledge for anyone serious about mastering CSS. The tool includes visual highlighting of affected elements and real-time preview of style changes, making it an interactive learning experience rather than a passive inspection tool.

JavaScript debugging capabilities include a fully functional console with command history, variable inspection, and breakpoint management. The console supports modern JavaScript features and provides intelligent autocomplete suggestions based on the execution context. Breakpoints can be set on lines, conditions, or DOM events, with detailed stack traces and scope inspection when execution pauses. This professional-grade debugging environment helps students understand program flow and develop debugging skills applicable to any programming language.

Network request monitoring captures all HTTP traffic between the browser and servers, displaying headers, payloads, timing information, and response data. This tool is essential for understanding how web applications communicate, how APIs work, and how to diagnose connectivity issues. The interface organizes requests logically and provides powerful filtering and search capabilities for navigating large request logs.

Technical Implementation

Hack Stuff is built using modern web technologies including Service Workers for offline functionality, Web Components for modular architecture, and CSS Custom Properties for theming. The application employs progressive enhancement strategies, ensuring that core functionality works across all modern browsers while providing enhanced features where supported. Performance optimization techniques including code splitting, lazy loading, and efficient caching ensure the tool remains responsive even on lower-powered devices common in educational settings.

Security architecture ensures that all tools operate within browser sandbox constraints, preventing access to sensitive system resources while maximizing available functionality. The sandbox implementation has been thoroughly audited to identify and close potential vulnerabilities, ensuring that students can experiment freely without risk of compromising their devices or data. All execution occurs locally within the browser, with no data transmitted to external servers unless explicitly requested by the user.

The modular architecture allows for easy extension with new tools and features. A documented API enables developers to create additional utilities that integrate seamlessly with the existing suite. This extensibility has fostered a community of contributors who regularly add new tools and improve existing ones, ensuring the suite remains current with evolving development practices and educational needs.

Educational Value

Hack Stuff serves as an exceptional educational resource for computer science curricula, providing hands-on experience with tools that students will encounter in professional settings. By working with these utilities in a safe, controlled environment, students develop practical skills that transfer directly to real-world development work. The platform supports self-directed learning while also serving as an effective classroom tool for demonstrations and exercises.

Teachers have incorporated Hack Stuff into courses covering web development, computer science fundamentals, and digital literacy. The tools support various teaching methodologies including guided exploration, problem-based learning, and project-based instruction. Detailed documentation and tutorial materials help educators integrate the platform into their curricula effectively.

Usage Guidelines

Access to certain advanced features may be restricted based on school network policies or device management configurations. The tool is designed to work within these constraints while still providing as much functionality as possible. Users should consult with their school's technology department if specific features are unavailable, as network administrators may be able to adjust settings to enable additional functionality for educational purposes.

Use of these tools should align with school policies and be limited to legitimate educational purposes. Responsible use of powerful tools is an important skill that students develop through their engagement with this platform.

Sawfish Game Portal

By Sawfish Developer Group

Category: Games Hub / Launcher

Overview

The Sawfish Game Portal represents a significant advancement in how students access and discover browser-based games within educational environments. Acting as a unified launcher and collection point for all approved games available through the Sawfish ecosystem, the portal eliminates the need for students to bookmark and manage dozens of individual game links. Instead, everything is available from one beautifully designed, intelligently organized interface that makes finding the perfect game for any moment effortless.

The portal's design philosophy centers on the belief that students deserve easy access to quality entertainment that enhances their breaks without compromising educational values. Every game in the collection has undergone careful review to ensure it meets school-appropriate standards, contains no excessive advertising, and operates efficiently on school hardware. This curation process ensures that the gaming experience is consistently positive and appropriate for the educational context.

Beyond simple game access, the portal provides sophisticated organization and discovery features that help students find games matching their interests and available time. The recommendation engine learns from play patterns to suggest new titles, while the achievement system tracks progress across all games in a unified view. Social features allow students to see what their friends are playing and share their favorite discoveries, creating a sense of community around the gaming experience.

Key Features and Capabilities

The sophisticated categorization system represents one of the portal's most valuable features. Games are organized by multiple classification schemes including genre (puzzle, platformer, simulation, arcade, educational), gameplay style (single-player, competitive, cooperative), session length (quick 5-minute games, extended sessions), and mood (relaxing, challenging, creative, competitive). This multi-dimensional organization helps students find appropriate games quickly regardless of their specific needs at any moment.

Each game card displays comprehensive information to help students make informed choices. Average playtime estimates help students select games that fit their available break period. Difficulty ratings provide transparency about the skill level required, preventing frustration from games that are too easy or too hard. Community ratings give social proof of quality, while category tags enable filtering by specific interests. Screenshots and brief descriptions provide additional context that helps students understand what they're getting into before committing to a game.

The unified achievement and progress system tracks accomplishments across all games in the collection. Students can see their total achievements, compare progress with friends, and work toward completion percentages for each game. This system adds long-term engagement to games that might otherwise be played once and forgotten, as students return to complete challenges and earn recognition for their accomplishments. Leaderboards for various achievements create friendly competition that motivates continued play.

Personalized recommendations analyze play history and preferences to suggest new games that align with individual tastes. The recommendation engine considers factors including preferred genres, session length patterns, achievement completion rates, and explicit user ratings. Recommendations are presented unobtrusively and can be easily dismissed if they don't interest the student, ensuring the feature enhances rather than intrudes upon the experience.

Game Curation Process

All games available through the portal have undergone a rigorous approval process designed to ensure educational appropriateness and quality standards. The curation team evaluates games across multiple dimensions including content safety (no violence, inappropriate themes, or mature content), advertising practices (no excessive ads, no deceptive monetization), educational value (potential for skill development or learning), and technical quality (performance, reliability, compatibility).

New game submissions enter a review queue where they are evaluated against these criteria. Submissions that meet all standards are added to the collection with full feature support. Games that nearly meet standards may be added with age restrictions or recommended session limits. Substantially non-compliant games receive detailed feedback explaining why they were not approved and what changes might enable future approval. This constructive approach encourages high-quality submissions while maintaining collection standards.

Existing games are periodically re-evaluated to ensure continued compliance, particularly after content updates from developers. Games that receive major updates undergo expedited re-review to verify that changes maintain safety standards. Students and teachers can report concerns about games, triggering investigation that may result in restriction or removal if issues are confirmed.

Performance and Compatibility

The portal has been optimized for the diverse hardware and network conditions found in educational environments. Games are tested across multiple device types including older Chromebooks, iPads, Windows laptops, and desktop computers to ensure broad compatibility. Network optimization ensures games load quickly even on bandwidth-constrained connections, with aggressive caching and intelligent prefetching minimizing wait times.

All game data syncs across devices through the Sawfish account system, so students can start a game on one device and continue on another without losing progress. This cross-device synchronization is particularly valuable in BYOD environments where students may use different devices at school and home. The sync system uses efficient delta updates to minimize bandwidth usage while ensuring data integrity.

Draw a Circle

By Sawfish Developer Group

Category: Games / Skill / Reflex

Overview

Draw a Circle stands as a masterclass in elegant game design, presenting players with an elegantly simple premise that hides surprising depth and endless replayability. The objective is deceptively straightforward: draw the most perfect circle possible using only your finger or mouse, with the game's sophisticated geometric analysis algorithms evaluating your creation across multiple precision metrics. What begins as a casual attempt at drawing quickly evolves into an addictive pursuit of perfection, with players returning repeatedly to improve their scores and master the subtle techniques required for circle-drawing excellence.

The game represents a perfect synthesis of accessibility and challenge. Players of any age or skill level can immediately understand the goal and begin playing, yet the path to mastery offers nearly unlimited room for improvement. This design philosophy ensures that the game remains engaging whether played casually during a short break or seriously by players pursuing world-leading scores. The learning curve is continuous, with small improvements in technique yielding measurable results that encourage continued practice.

Beyond its entertainment value, Draw a Circle serves as an engaging tool for developing fine motor control, patience, and the understanding of geometric precision. The game's mathematical foundation provides genuine insight into the nature of circularity, helping players develop intuitive understanding of concepts from geometry and physics. Educational applications range from elementary students learning about shapes to professionals seeking to improve handwriting or drawing skills.

Game Mechanics and Features

The geometric analysis system evaluates drawn circles across multiple precision dimensions including deviation from mathematical circularity, consistency of stroke width, smoothness of curves, and absence of jagged edges or corrections. The algorithm compares the drawn path against a perfect circle of the same diameter, measuring point-by-point deviation and calculating aggregate precision scores. This multi-dimensional analysis provides nuanced feedback that helps players understand exactly where their drawing technique needs improvement.

Multiple difficulty modes cater to different skill levels and purposes. Casual mode provides visual guides and forgiving scoring that encourages experimentation and learning. Standard mode removes assistance while maintaining accessible scoring thresholds. Expert mode removes all guides and implements strict scoring that challenges even skilled players. Speed mode adds time pressure, requiring players to draw perfect circles under time constraints. Each mode maintains separate leaderboards, ensuring fair competition across different skill levels.

Daily challenges provide fresh content and community engagement opportunities. Each day presents a specific drawing challenge, such as drawing circles of a particular size range or achieving a minimum precision score within a limited number of attempts. Global leaderboards show how your performance compares to other players worldwide, creating friendly competition and motivation to improve. Weekly and monthly challenges aggregate daily results to identify consistent high performers.

Training mode offers interactive lessons on drawing technique, breaking down the process into learnable components. Players practice individual skills including smooth stroke generation, consistent pressure application, and steady hand control. Immediate feedback helps players identify specific weaknesses in their technique, while progress tracking shows improvement over time. This structured approach to skill development transforms circle-drawing from an intuitive art into an improvable science.

Technical Implementation

The drawing system employs advanced input processing that works consistently across different input methods including touch screens, trackpads, and mouse devices. Input smoothing algorithms remove jitter from touch input while preserving the genuine characteristics of the user's movement. The system captures input at high frequency to ensure no meaningful motion data is lost, while computational efficiency ensures immediate feedback without noticeable latency.

The scoring algorithm has been refined through extensive testing and calibration against human assessments of circle quality. The algorithm's outputs correlate strongly with human perception of circularity, ensuring that scores feel fair and intuitive. Mathematical optimization ensures the algorithm runs efficiently even on lower-powered devices, while parallel processing on capable devices enables additional precision analysis.

Progress is saved locally on each device, with optional cloud sync for students who want to maintain their records across devices or compete globally. The local storage system efficiently tracks thousands of historical attempts, providing rich data for personal progress analysis. Privacy-respecting analytics provide aggregate information about population-wide performance without exposing individual data.

Usage in Educational Settings

Draw a Circle has found surprising applications in educational contexts beyond entertainment. Occupational therapists use it to assess and develop fine motor control in patients. Art teachers incorporate it as a foundational exercise for developing drawing skills. Geometry teachers use it to demonstrate mathematical concepts related to circles and precision. The game's immediate feedback and clear metrics provide objective assessment of skills that would otherwise be subjective and difficult to measure.

The game's minimal resource requirements make it accessible in any educational technology environment. Fast loading times ensure it can be launched even on restricted networks, while offline functionality enables use in locations with limited connectivity. The straightforward gameplay requires minimal instruction, allowing valuable class time to remain focused on learning objectives rather than technical setup.

2048

By Sawfish Developer Group

Category: Games / Puzzle / Strategy

Overview

2048 has earned its place as one of the most influential puzzle games of the modern era, combining elegant mathematical concepts with addictive gameplay that has captivated millions of players worldwide. The objective is elegantly simple: combine matching numbered tiles to reach the elusive 2048 tile. However, beneath this straightforward premise lies substantial strategic depth that rewards planning, pattern recognition, and adaptive thinking. The game has become a cultural phenomenon, spawning countless variations, academic studies, and dedicated communities of enthusiasts pursuing mastery.

This browser-optimized implementation preserves the core experience that made the original game successful while adding features specifically valuable in educational contexts. The game serves as an engaging introduction to mathematical concepts including powers of two, exponential growth, and probability, all within an accessible and entertaining framework. Players develop intuitive understanding of these concepts through repeated engagement, often without realizing they are learning.

The game design exemplifies how simple rules can generate complex emergent behavior. Each move affects the entire board state, creating cascading consequences that can enable spectacular combinations or lead to unexpected dead ends. This complexity emerges naturally from the interaction of simple rules, making it an excellent example of complex systems in action. Students interested in mathematics, computer science, or game design can find endless fascination in analyzing the game's underlying structure.

Game Features and Variations

Touch-optimized controls make swiping on tablets and touchscreens feel natural and responsive. The control system interprets gestures with high accuracy, distinguishing between deliberate moves and accidental input. Visual feedback during swipes clearly indicates the intended direction, preventing confusion about which direction the tiles will move. Keyboard controls remain available for desktop users, with support for both arrow keys and WASD.

Multiple board sizes extend the classic 4x4 experience with more challenging variants. The 5x5 board reduces tile density, requiring longer-term planning and providing more room for complex tile arrangements. The 6x6 board further increases complexity, challenging even experienced players to develop new strategies. Each size maintains separate high scores and achievement tracking, allowing players to specialize in their preferred challenge level.

The undo feature allows players to backtrack mistakes, providing a safety net that encourages experimentation with different strategies. This feature is particularly valuable for educational contexts where the learning process is more important than achieving high scores. Players can explore alternative moves and see their consequences without permanently losing progress, enabling deeper understanding of cause and effect in the game's system.

The hint system provides subtle suggestions when players become stuck, offering guidance without revealing optimal moves. Hints are designed to prompt strategic thinking rather than replace it, highlighting general principles or pointing out opportunities without specifying exact moves. This graduated assistance helps players develop their own skills rather than creating dependency on external guidance.

Strategic Considerations

Successful 2048 play requires balancing immediate tile placement with long-term board management. Players must anticipate how the board will evolve over multiple moves, positioning tiles to enable future combinations while avoiding situations that block progress. The corner strategy, which involves concentrating high-value tiles in a single corner, has become a standard technique that provides consistent results for players at all levels.

Understanding tile spawning probabilities helps players make informed decisions about which moves to prioritize. New tiles spawn in random empty positions, with a 90% chance of being a 2 and a 10% chance of being a 4. This statistical reality affects how players manage board space and prioritize tile placement. Advanced players develop probabilistic intuitions about where new tiles are likely to appear and plan accordingly.

The game provides an excellent framework for developing planning and foresight skills. Players must consider multiple moves ahead, evaluating how current decisions will affect future options. This type of strategic thinking transfers to real-world decision making, making the game valuable beyond its entertainment function. Educational applications include teaching concepts from mathematics, economics, and life planning.

Educational Applications

Teachers have found 2048 valuable for demonstrating mathematical concepts in engaging ways. The powers of two progression (2, 4, 8, 16, 32, 64, 128, 256, 512, 1024, 2048) provides concrete visualization of exponential growth. Students can observe and count tiles to understand how quickly values compound. The game naturally introduces concepts including place value, addition, and pattern recognition.

Analysis of optimal play involves concepts from game theory, probability, and optimization. Students interested in computer science can explore algorithms for automated play, from simple heuristics to sophisticated machine learning approaches. The game's complexity provides meaningful challenges while remaining accessible enough for classroom exploration.

Overview

Minecraft Web brings the legendary sandbox building experience directly to browser environments, enabling students to explore, build, and create within the iconic blocky universe without requiring downloads, installations, or high-end hardware. This specialized web port captures the essence that made Minecraft a global phenomenon while adapting the experience for the unique constraints and opportunities of browser-based gaming. Players can dive into procedurally generated worlds filled with diverse biomes, resources, and opportunities for creative expression.

The game's open-ended design philosophy provides unlimited creative freedom within a structured world. Players choose their own goals, whether constructing monumental architectures, designing intricate redstone contraptions, exploring vast cave systems, or simply surviving against environmental challenges. This player-driven experience contrasts with the goal-oriented structure of most games, making Minecraft uniquely suited to diverse play styles and preferences.

The educational potential of Minecraft has been widely recognized, with the game used in classrooms worldwide to teach subjects ranging from history and geography to mathematics and computer science. The web version maintains this educational value while adding accessibility benefits that enable use in environments where traditional Minecraft installation is not possible.

Game Features and Capabilities

The Beta version introduces rendering engines specifically optimized for web performance, achieving smooth frame rates even on Chromebooks and older laptops that would struggle with the full desktop version. Advanced techniques including level-of-detail management, efficient chunk loading, and intelligent culling ensure that system resources are focused on visible, relevant content. These optimizations enable a genuine Minecraft experience on hardware previously considered incapable of running the game.

Procedural terrain generation creates diverse worlds with varied biomes including forests, deserts, mountains, oceans, and underground cave systems. Each world seed produces a unique landscape, ensuring that no two adventures are exactly alike. The generation algorithms balance exploration rewards with spawn safety, giving players interesting environments to discover without immediate danger at spawn points.

The inventory and crafting systems have been streamlined for browser and touch input. Drag-and-drop mechanics, contextual crafting recipes, and simplified interfaces make resource management intuitive even on touch screens. Essential crafting and building operations can be completed with minimal clicks, reducing friction between creative vision and execution.

Creative mode provides unlimited resources and the ability to fly, enabling ambitious building projects without resource gathering constraints. Players can focus entirely on construction and design, experimenting with architectural ideas and artistic expressions. Survival mode challenges players to gather resources, establish shelters, and manage health and hunger while exploring and building.

Important Usage Notes

Critical Information About Single Player Mode

IMPORTANT: In this web version, single player mode does not include crafting functionality. You MUST use a multiplayer server to access full crafting features. This limitation exists because the web implementation requires server-side processing for certain game mechanics that are normally handled client-side in the desktop version.

For the best experience, we strongly recommend joining the official first server listed when launching the application. This server provides the most stable multiplayer experience, active player community, and full access to all crafting and building features. Single player functionality is being actively developed, and this limitation may be addressed in future updates.

When playing on multiplayer servers, all progress and builds are persistently saved to the server, ensuring that your creations are preserved between sessions. The server also enables collaboration with other players, making it ideal for group projects and social building experiences.

Educational Applications

Minecraft has established itself as a powerful educational tool used in classrooms worldwide. History teachers recreate historical sites for virtual tours. Geography students study terrain and climate patterns. Mathematics concepts like volume, area, and symmetry become tangible through building projects. Computer science principles can be explored through redstone circuitry and command block programming.

The web version enables these educational applications in environments where traditional Minecraft installation is not possible. Schools with managed devices, restricted software policies, or older hardware can now provide Minecraft-based learning experiences. The accessibility improvements significantly expand the potential reach of Minecraft education.

Block Blast delivers an addictive puzzle experience that challenges spatial reasoning and strategic planning through its elegant block placement mechanics.
